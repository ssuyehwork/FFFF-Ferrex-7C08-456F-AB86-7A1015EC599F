## 我给它取名：

# **Ferrex**

---

### 命名逻辑

| 来源 | 含义 |
|---|---|
| **Fer**rum（拉丁语：铁） | 指 NTFS 的底层金属质感，原始、硬核、直接读磁盘 |
| **Rex**（拉丁语：王） | 同类工具中的王者 |
| **Ferrex** 合体 | "铁王" — 不靠任何中间层，直接掌控文件系统的王 |

---

### 为什么不是其他名字

- ~~FastFind~~ → 太普通
- ~~NTFScan~~ → 太技术，不够霸气
- ~~Indexer~~ → 像工具库不像产品
- **Ferrex** → 短、硬、能注册商标、英语母语者也能一眼记住

---

### 品牌延伸

```
Ferrex          → 主程序
ferrex index    → 建立索引
ferrex search   → 搜索
ferrex watch    → 监听模式
ferrex.idx      → 索引文件后缀

口号：
"No database. No compromise. Just iron speed."
（无数据库，无妥协，唯铁速。）
```
